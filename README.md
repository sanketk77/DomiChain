## 🧪 Key Features

- 🔐 MetaMask wallet connection
- 🏘️ Add properties to the blockchain (owner-only)
- 🧾 View real-time listed properties
- 🔍 Search & filter (by name/city)
- 🧭 Navigate to property details (`/property/:id`)
- 💻 Fully responsive UI (mobile + desktop)
- 🔜 Upcoming: Map view, favorites, edit property, booking system
